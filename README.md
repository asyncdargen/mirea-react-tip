# MIREA React Tip

Простое React + TypeScript приложение — расчитывает КПС.

<img src="https://github.com/asyncdargen/mirea-react-tip/raw/main/example/example.png" alt="Пример использования" width="600" />

## Краткое описание
Это учебный проект: небольшой интерфейс для расчёта кликов в секунду.

## Доступность
Сайт развернут и доступен онлайн на Vercel: https://mirea-react-tip.vercel.app/

## Основной функционал
- Расчет КПС
- Общее кол-во кликов
- Кнопка сброса.

## Технологии
- React
- TypeScript
- CSS (модули или простой CSS)
- Сборка: Vite / Create React App (зависит от шаблона в репозитории)

## Запуск (локально)
Требования: Node.js 16+ и npm или yarn.

1. Клонировать репозиторий:
   git clone https://github.com/asyncdargen/mirea-react-tip.git
2. Перейти в каталог проекта:
   cd mirea-react-tip
3. Установить зависимости:
   npm install
   или
   yarn
4. Запустить приложение:
   npm start
   или
   yarn start
5. Открыть в браузере: http://localhost:3000 (или адрес, который выдаст dev-сервер)
