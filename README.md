# Генератор паролей — mirea-react-tip

Небольшое React + TypeScript приложение: генератор случайных паролей. Приложение генерирует 5 паролей за один клик и позволяет копировать каждый пароль в буфер обмена.

## Что делает
- Генерирует 5 паролей одновременно.
- Настройки генерации: длина пароля, включать строчные/заглавные буквы, цифры и символы.
- Кнопки «Копировать» рядом с каждым паролем (используется `navigator.clipboard` с фолбеком через textarea).
- Генерация с помощью криптографически стойкого источника случайных чисел (`window.crypto.getRandomValues`).

## Быстрый старт (локально)
Требования: Node.js 16+ и npm.

1. Клонировать репозиторий:

```bash
git clone https://github.com/asyncdargen/mirea-react-tip.git
cd mirea-react-tip
```

2. Установить зависимости и запустить dev-сервер (Vite):

```bash
npm install
npm run dev
```

3. Откройте в браузере адрес, который выведет Vite (по умолчанию `http://localhost:5173`).

4. Для сборки на продакшн:

```bash
npm run build
npm run preview
```

## Файлы, которые важны
- `src/App.tsx` — основной компонент генератора паролей (логика и UI).
- `src/index.css` — стили интерфейса.

## Технические замечания
- Копирование в буфер работает в безопасном контексте (https/localhost). При отсутствии доступа к `navigator.clipboard` используется стандартный фолбек.
- Для генерации паролей применяется `window.crypto.getRandomValues`, что даёт более предсказуемо стойкие случайные значения по сравнению с `Math.random()`.
